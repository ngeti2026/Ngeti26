# NGETI 2026 - International Conference on Next Generation Engineering

## Conference Info

**Date**: December 4-5, 2026
**Venue**: Viswajyothi College of Engineering and Technology, Kerala
**Theme**: AI, Computing, Communication, Power Systems, and Sustainable Infrastructure

## About

NGETI 2026 is an IEEE co-sponsored international conference bringing together researchers and practitioners to share insights in next-generation engineering technologies.

## Development

### Info
This project is built with:
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

### Setup
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
